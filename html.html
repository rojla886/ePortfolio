🔥 HTML for Beginners – Complete Guide

Learn HTML from scratch: every tag explained, live demos, and code snippets. No experience needed.

1. What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a webpage using elements (tags). Think of it as the skeleton of a website – it holds everything together.

🌐 Live fact: This page you are reading right now is built with HTML. Right-click anywhere and select “View Page Source” to see the actual HTML code.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
💡 HTML uses tags inside angle brackets: <p> (opening) and </p> (closing). Most tags come in pairs.

2. Document Structure (The Skeleton)

<!DOCTYPE html>

Declares that the document is HTML5 – must be the very first line.

<html>

Root element that wraps all page content.

<head>

Contains meta information (title, character set, CSS links) – not visible on the page.

<body>

Contains everything visible: text, images, buttons, etc.
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Awesome Page</title>
</head>
<body>
    <h1>Welcome!</h1>
    <p>Visible content.</p>
</body>
</html>

4. Lists (Organising Information)

Unordered list (<ul> + <li>)

Bullet points – order doesn’t matter.
  • Apples
  • Bananas
  • Oranges
<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Oranges</li>
</ul>

Ordered list (<ol> + <li>)

Numbered list – sequence matters (steps, rankings).
  1. Wake up
  2. Brush teeth
  3. Eat breakfast
<ol>
  <li>Wake up</li>
  <li>Brush teeth</li>
  <li>Eat breakfast</li>
</ol>

Description list (<dl>, <dt>, <dd>)

Terms and definitions (glossary).

Structures the page.

  <dt>HTML</dt>
  <dd>Structures the page.</dd>
  <dt>CSS</dt>
  <dd>Styles the page.</dd>
</dl>

6. Images & Multimedia

<img> (Image)

Self‑closing tag. Requires src (source) and alt (alternative text for accessibility).
<img src="photo.jpg" alt="Description of image" width="300">

<audio> and <video>


<audio controls>
  <source src="song.mp3" type="audio/mpeg">
</audio>
<video width="250" controls>
  <source src="movie.mp4" type="video/mp4">
</video>

10. Div & Span – Grouping Elements

<div> is a block‑level container; <span> is inline. They are used with class and id for styling and scripting.
This is a <div> (block).
This is a <span> (inline).

You can target elements by class or id.

<span id="special">...</span>

11. HTML Entities (Special Characters)

Entities represent characters that would otherwise be interpreted as HTML code, like < and >.
< (less than) → &lt;
> (greater than) → &gt;
& (ampersand) → &amp;
© (copyright) → &copy;
♥ (heart) → &hearts; 
&lt;  →  <
&gt;  →  >
&amp;  →  &
&copy; → ©
&hearts; → ♥

12. Iframes (Embed External Content)

Embed another webpage or a video inside your page.
<iframe src="https://example.com" width="400" height="300"></iframe>

13. Meta Tags (Information for Browsers & SEO)

Meta tags go inside <head>. They provide metadata like description, keywords, viewport settings.
<meta charset="UTF-8">
<meta name="description" content="Learn HTML easily">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
🔍 The viewport meta tag is essential for responsive design on mobile devices.

16. HTML Best Practices for Beginners

  • ✅ Always include <!DOCTYPE html> and <meta charset="UTF-8">
  • ✅ Use semantic tags (<header>, <nav>, <main>) instead of too many <div>s
  • ✅ Add alt attributes to all images
  • ✅ Keep heading hierarchy logical (<h1> → <h2> → etc.)
  • ✅ Close all tags (except self‑closing like <img>)
  • ✅ Use lower case for tag names and attributes
  • ✅ Test your HTML with the W3C Validator
