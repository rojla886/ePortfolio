Chapter 01

Understanding the Web Skeleton

HTML (**HyperText Markup Language**) is the standard markup language used to create the structure of web pages. Unlike programming languages that focus on logic, HTML focuses on content organization.

How it works:

HTML uses "Tags" (elements wrapped in angle brackets <>) to tell the browser what type of content is being displayed. <h1> tells the browser "This is a big title," while <p> means "This is a paragraph."

<p>Welcome to the Web!</p>
Chapter 02

Document Structure (Boilerplate)

Every single HTML page on the internet must follow a strict, mandatory skeleton to be valid.

  • !DOCTYPE html: Informs the browser that this is an HTML5 document.
  • html: The root element that wraps all content.
  • head: A container for metadata (stuff the user doesn't see, like styles and SEO).
  • body: Where all visible text, images, and buttons live.
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Professional Site</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

Chapter 05

Navigation & Hyperlinks

The "H" in HTML stands for **HyperText**, meaning links. The <a> tag uses the href attribute to define the destination.

<!-- External Link -->
<a href="https://google.com" target="_blank">Visit Google</a>

<!-- Email Link -->
<a href="mailto:help@site.com">Email Us</a>

Chapter 16

Industry Best Practices

  • Always use lowercase for tags.
  • Always include alt on images.
  • Use 2 or 4 spaces for indentation.
  • Only use one <h1> per page.
  • Always close your tags.
  • Use UTF-8 Character Encoding.
