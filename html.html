Chapter 01

What is HTML?

HTML (**HyperText Markup Language**) is the most basic building block of the Web. It defines the meaning and structure of web content. It's not a programming language, but a markup language that uses tags to annotate text, images, and other content for display in a Web browser.

Why learn it? Every single website, from Google to Facebook, is built on a foundation of HTML. Without it, the web would be plain text.

Chapter 02

The Standard Skeleton

Every HTML file must follow this exact document structure to be rendered correctly by browsers.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>
