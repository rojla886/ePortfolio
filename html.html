1. What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It describes the structure of a webpage using elements (tags). Think of it as the skeleton of a website – it holds everything together.

🌐 Live fact: This page you are reading right now is built with HTML. Right-click anywhere and select "View Page Source" to see the actual HTML code.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
💡 HTML uses tags inside angle brackets: <p> (opening) and </p> (closing). Most tags come in pairs.

2. Document Structure (The Skeleton)

<!DOCTYPE html>

Declares that the document is HTML5 – must be the very first line.

<html>

Root element that wraps all page content.

<head>

Contains meta information (title, character set, CSS links) – not visible on the page.

<body>

Contains everything visible: text, images, buttons, etc.
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Awesome Page</title>
</head>
<body>
    <h1>Welcome!</h1>
    <p>Visible content.</p>
</body>
</html>

Headings define titles and subtitles. <h1> is most important, <h2> to <h6> are subheadings. Search engines use them for SEO.

Defines a paragraph. Browsers add automatic spacing before and after.

4. Lists (Organising Information)

Unordered list (<ul> + <li>)

Bullet points – order doesn't matter.
  • Apples
  • Bananas
  • Oranges
<ul>
  <li>Apples</li>
  <li>Bananas</li>
</ul>

Ordered list (<ol> + <li>)

Numbered list – sequence matters.
  1. Wake up
  2. Brush teeth
  3. Eat breakfast
<ol>
  <li>Wake up</li>
  <li>Brush teeth</li>
</ol>

Description list (<dl>, <dt>, <dd>)

Terms and definitions (glossary).

Structures the page.
CSS
Styles the page.
<dl>
  <dt>HTML</dt>
  <dd>Structures the page.</dd>
</dl>

12. Iframes (Embed External Content)

Embed another webpage or a video inside your page.
<iframe src="https://example.com" width="400" height="300"></iframe>

13. Meta Tags (Information for Browsers & SEO)

Meta tags go inside <head>. They provide metadata like description, keywords, and viewport settings.
<meta charset="UTF-8">
<meta name="description" content="Learn HTML easily">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
🔍 The viewport meta tag is essential for responsive design on mobile devices.

15. HTML5 Form Validation (No JavaScript Needed)

HTML5 provides built-in validation: required, pattern, min/max, email, etc.






Try submitting without filling – browser will show a validation message.

<input type="text"   placeholder="Your name" required>
<input type="email"  placeholder="Email"     required>
<input type="number" min="18" max="99"       required>
<input type="submit" value="Submit">

16. HTML Best Practices for Beginners

  • ✅ Always include <!DOCTYPE html> and <meta charset="UTF-8">
  • ✅ Use semantic tags (<header>, <nav>, <main>) instead of too many <div>s
  • ✅ Add alt attributes to all images
  • ✅ Keep heading hierarchy logical (<h1> → <h2> → etc.)
  • ✅ Close all tags (except self-closing like <img>)
  • ✅ Use lower case for tag names and attributes
  • ✅ Test your HTML with the W3C Validator
