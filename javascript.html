Chapter 01

What is JavaScript?

JavaScript (JS) is the programming language that makes websites interactive and dynamic. While HTML gives structure and CSS provides styling, JavaScript brings behaviour – handling button clicks, validating forms, updating content without reloading, fetching data from servers, creating animations, and much more.

💡 Why learn JavaScript? It is essential for front‑end development, enables full‑stack development (with Node.js), has a huge ecosystem (React, Vue, Angular), and offers excellent career opportunities.

Click the button below – JavaScript changes the text and style instantly!

// Basic example: change content and style with JavaScript
document.getElementById('welcomeBtn').addEventListener('click', () => {
  const msg = document.getElementById('welcomeMessage');
  msg.textContent = 'Hello! JavaScript is working 🎉';
  msg.style.color = '#0ea5e9';
});

📚 Core JavaScript Concepts

Concept 01

let, const, and var

Variables store data. Use let for values that change, const for values that never change (objects/arrays declared with const can still be modified, but not reassigned). Avoid var (old, function‑scoped). Data types: strings, numbers, booleans, arrays, objects. 
let age = 25;      age = 26;        // allowed
const name = 'John'; // name = 'Jane' → Error
const tasks = [];   // tasks.push('new') → allowed (object mutation)
Concept 02

Functions (Reusable Code)

Functions encapsulate reusable logic. Can take parameters and return values. Arrow functions (() => {}) provide concise syntax. 
// Function declaration
function greet(name) {
  return `Hello ${name}`;
}
// Arrow function
const add = (a, b) => a + b;
Concept 03

Event Listeners & DOM Manipulation

addEventListener waits for user actions (click, input, submit). DOM manipulation: createElement, appendChild, classList – dynamically update the page without reload. 
document.getElementById('domDemoBtn').addEventListener('click', () => {
  const newDiv = document.createElement('div');
  newDiv.textContent = '🎉 New element added!';
  newDiv.style.background = '#e2e8f0';
  document.getElementById('domDemoContainer').appendChild(newDiv);
});
Concept 04

Array Methods (push, filter, forEach, map)

Powerful methods to manipulate lists: push (add), filter (select), forEach (loop), map (transform). 
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(n => n * 2);   // [2,4,6,8]
const evens = numbers.filter(n => n % 2 === 0); // [2,4]
numbers.forEach(n => console.log(n));
Concept 05

LocalStorage (Persistent Data)

Stores key‑value pairs in the browser. Data survives page refresh. Use JSON.stringify() to store objects/arrays, JSON.parse() to retrieve. 
// Save
localStorage.setItem('tasks', JSON.stringify(tasks));
// Load
const saved = localStorage.getItem('tasks');
if (saved) tasks = JSON.parse(saved);
Concept 06

Promises, Async/Await & Fetch

Promises handle asynchronous operations. Async/await is modern syntax that makes promises easier. Fetch API makes HTTP requests. 
async function fetchUser() {
  try {
    const response = await fetch('https://jsonplaceholder.typicode.com/users/1');
    const user = await response.json();
    console.log(user.name);
  } catch (error) {
    console.error('Failed:', error);
  }
}
Concept 07

Error Handling (try...catch)

try...catch prevents scripts from crashing when an error occurs. Essential for network requests and user input. 
try {
  const invalidJSON = '{ name: "John" }'; // missing quotes
  const data = JSON.parse(invalidJSON);
} catch (error) {
  console.error('Caught:', error.message);
}
Concept 08

Classes (Object‑Oriented Programming)

Classes are blueprints for creating objects. Constructor initialises properties, methods define behaviour. 
class Task {
  constructor(text) {
    this.id = Date.now();
    this.text = text;
    this.completed = false;
  }
  toggle() {
    this.completed = !this.completed;
  }
}
const myTask = new Task('Learn OOP');
Concept 09

Template Literals (Backticks & ${})

Use backticks ` and ${expression} to embed variables and expressions in strings – cleaner than concatenation. 
const completed = 5, total = 10;
const message = `${completed} of ${total} tasks done (${(completed/total)*100}%)`;
// Multi‑line HTML
const html = `
  <div class="card">
    <h3>${title}</h3>
  </div>
`;
