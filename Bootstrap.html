🎨 Bootstrap 5 Complete Guide

Grid • Components • Utilities • Forms • JavaScript Plugins • Every concept with live demo + code
What is Bootstrap? Bootstrap is the world's most popular front‑end framework. It provides a responsive 12‑column grid system, hundreds of pre‑styled components (cards, modals, navbars, carousels…), utility classes for spacing/alignment, and JavaScript plugins. With Bootstrap, you build professional, mobile‑friendly websites fast – without writing custom CSS for every element.

Bootstrap 5 Components

  • JavaScript plugins (modals, carousels, tooltips, toasts…)
  • Utility classes (spacing, flex, display, typography)

📐 1. Grid System (12 columns)

Bootstrap’s grid is built with flexbox. Use .container (or .container-fluid), .row, and .col-* classes. Columns add up to 12. Breakpoints: col-sm, col-md, col-lg, col-xl, col-xxl.
.col-md-4
.col-md-4
.col-md-4
.col-6 .col-md-3
.col-6 .col-md-3
.col-6 .col-md-3
.col-6 .col-md-3
<div class="container">
  <div class="row">
    <div class="col-md-4">...</div>
    <div class="col-md-4">...</div>
    <div class="col-md-4">...</div>
  </div>
</div>
💡 Columns automatically wrap when they exceed 12. Use g-3 for gutters (gap between columns).

🧩 2. Bootstrap Components

📂 Accordion

Vertically collapsing panels. Great for FAQs or sections where you want to save space. Each panel toggles independently.

Content for item 1.

Content for item 2.
<div class="accordion" id="myAccordion">
  <div class="accordion-item">
    <h2 class="accordion-header">
      <button class="accordion-button" data-bs-toggle="collapse" data-bs-target="#collapse1">Title</button>
    </h2>
    <div id="collapse1" class="accordion-collapse collapse show" data-bs-parent="#myAccordion">
      <div class="accordion-body">Content</div>
    </div>
  </div>
</div>

💬 Tooltips & Popovers

Tooltips appear on hover; popovers are clickable cards. Must be initialised with JavaScript.
<button data-bs-toggle="tooltip" title="Tooltip">Hover</button>
<button data-bs-toggle="popover" title="Title" data-bs-content="Content">Click</button>
<script>
  var tooltips = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]')).map(t => new bootstrap.Tooltip(t));
  var popovers = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]')).map(p => new bootstrap.Popover(p));
</script>

📝 3. Forms & Validation

Bootstrap styles all form controls (.form-control, .form-select). Use .needs-validation and custom JavaScript for real‑time validation.
Required.
Valid email required.
<form class="needs-validation" novalidate>
  <div class="form-floating mb-3">
    <input type="text" class="form-control" required>
    <label>Name</label>
    <div class="invalid-feedback">Required</div>
  </div>
  <button type="submit">Submit</button>
</form>
