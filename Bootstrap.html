Chapter 01

What is Bootstrap?

Bootstrap is the world's most popular front‑end framework. It provides a responsive 12‑column grid system, hundreds of pre‑styled components, utility classes, and JavaScript plugins. With Bootstrap, you build professional, mobile‑friendly websites fast – without writing custom CSS for every element.

💡 Why Bootstrap? Saves development time, ensures cross‑browser consistency, and is fully responsive out of the box.

Bootstrap 5 Features
  • Responsive grid system (12 columns, 6 breakpoints)
  • Pre‑styled components
  • JavaScript plugins
  • Utility classes
  • Customizable via Sass
Chapter 02

Grid System (12 columns)

Bootstrap’s grid uses flexbox. Use .container, .row, and .col-* classes. Columns sum to 12. Breakpoints: sm, md, lg, xl, xxl.
.col-md-4
.col-md-4
.col-md-4
.col-6 .col-md-3
.col-6 .col-md-3
.col-6 .col-md-3
.col-6 .col-md-3
<div class="container"><div class="row"><div class="col-md-4">...</div></div></div>
💡 Use g-3 for gutters. Columns auto‑wrap when exceeding 12.
