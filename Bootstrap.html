Chapter 01

What is Bootstrap?

Bootstrap is the world's most popular front‑end framework. It provides a responsive 12‑column grid system, hundreds of pre‑styled components (cards, modals, navbars, carousels…), utility classes for spacing/alignment, and JavaScript plugins. With Bootstrap, you build professional, mobile‑friendly websites fast – without writing custom CSS for every element.

💡 Why Bootstrap? Saves development time, ensures cross‑browser consistency, and is fully responsive out of the box. Used by millions of websites. 
<!-- Include Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">
<!-- Include Bootstrap JS Bundle (includes Popper) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
Bootstrap 5 Features
  • Responsive grid system (12 columns, 6 breakpoints)
  • Pre‑styled components (cards, buttons, navbars, alerts…)
  • JavaScript plugins (modals, carousels, tooltips, toasts…)
  • Utility classes for spacing, flex, display, typography
  • Customizable via Sass variables
Chapter 02

Grid System (12 columns)

Bootstrap’s grid is built with flexbox. Use .container (or .container-fluid), .row, and .col-* classes. Columns sum to 12. Breakpoints: col-sm, col-md, col-lg, col-xl, col-xxl.
.col-md-4
.col-md-4
.col-md-4
.col-6 .col-md-3
.col-6 .col-md-3
.col-6 .col-md-3
.col-6 .col-md-3
<div class="container">
  <div class="row">
    <div class="col-md-4">...</div>
    <div class="col-md-4">...</div>
    <div class="col-md-4">...</div>
  </div>
</div>
💡 Columns automatically wrap when they exceed 12. Use g-3 for gutters (gap between columns).

Component 06

💬 Tooltips & Popovers

Tooltips appear on hover; popovers are clickable cards. Must be initialised with JavaScript.
<button data-bs-toggle="tooltip" title="Tooltip">Hover</button>
<button data-bs-toggle="popover" title="Title" data-bs-content="Content">Click</button>
<script>
  var tooltips = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]')).map(t => new bootstrap.Tooltip(t));
  var popovers = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]')).map(p => new bootstrap.Popover(p));
</script>

Chapter 03

Forms & Validation

Bootstrap styles all form controls (.form-control, .form-select). Use .needs-validation and custom JavaScript for real‑time validation.
Required.
Valid email required.
<form class="needs-validation" novalidate>
  <div class="form-floating mb-3">
    <input type="text" class="form-control" required>
    <label>Name</label>
    <div class="invalid-feedback">Required</div>
  </div>
  <button type="submit">Submit</button>
</form>
