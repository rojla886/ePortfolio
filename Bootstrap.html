What is Bootstrap?

Bootstrap is a free, open‑source CSS framework that provides:

  • A responsive 12‑column grid system – adapts to any screen size.
  • Pre‑styled components – navbar, cards, modals, forms, buttons, alerts, etc.
  • Utility classes – for spacing, typography, flexbox, display, colours.
  • JavaScript plugins – tooltips, popovers, carousels, offcanvas – no custom JS needed. 
<!-- Include CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">
<!-- Include JS bundle -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

📐 Bootstrap Grid System (12 columns)

The grid is the foundation of Bootstrap layouts. It uses containers, rows, and columns.

.col-md-4
.col-md-4
.col-md-4
.col-6 .col-md-3
.col-6 .col-md-3
.col-6 .col-md-3
.col-6 .col-md-3
📘 How it works: Columns are placed inside rows (.row). Each column has a width defined by numbers that add up to 12. .col-md-4 means "4 out of 12 columns on medium screens (≥768px)". .col-6 .col-md-3 means "6 columns on extra small screens, 3 columns on medium and up". 
<div class="container">
  <div class="row">
    <div class="col-md-4">Column 1</div>
    <div class="col-md-4">Column 2</div>
    <div class="col-md-4">Column 3</div>
  </div>
</div>
