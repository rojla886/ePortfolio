📌 1. Three Ways to Add CSS (Inline, Internal, External)

CSS can be added to HTML in three different ways. Each has its own purpose and use case.

1.1 Inline CSS (inside an HTML element)

Written directly inside an HTML tag using the style attribute. Affects only that single element.

This paragraph is styled with inline CSS.

This div also uses inline CSS. 
<p style="color: blue; font-weight: bold;">Text</p>
⚠️ Use inline CSS only for one‑off overrides. It mixes content with styling and is hard to maintain.

1.2 Internal CSS (inside <style> tag in <head>)

Written inside a <style> tag in the <head>. Applies to the entire HTML page.

This paragraph uses an internal CSS class.

Another internal CSS styled box
<style>
  .internal-demo { color: green; }
</style>
✅ Internal CSS is good for single‑page demos. For multi‑page sites, external CSS is better.

1.3 External CSS (separate .css file)

Stored in a separate .css file and linked via <link>. Best practice for real websites. 
<!-- In HTML head -->
<link rel="stylesheet" href="styles.css">

/* styles.css */
body { font-family: Arial; background: #f0f0f0; }
✅ External CSS keeps HTML clean, allows caching, and is reusable across many pages.

⚖️ 3. Specificity, Cascade & Inheritance

Specificity determines which rule wins when multiple rules target the same element. Order: !important > Inline > ID > Class/attribute/pseudo-class > Element.
This text changes colour based on selector priority.
/* Specificity hierarchy */
#id { }      /* Highest (100) */
.class { }   /* Medium (10) */
element { }  /* Lowest (1) */
💡 Use classes instead of IDs for styling to keep specificity low and maintainable.

📦 4. CSS Variables (Custom Properties)

Variables store values for reuse. Define them in :root (global) or inside a selector.
Background set via variable
:root {
  --primary-color: #2563eb;
}
button {
  background: var(--primary-color);
}

✅ 11. CSS Best Practices

  • ✔ Use external CSS for real projects.
  • ✔ Prefer classes over IDs for styling (lower specificity).
  • ✔ Use transform and opacity for smooth animations (GPU accelerated).
  • ✔ Always include box-sizing: border-box; globally.
  • ✔ Write responsive styles with @media queries.
  • ✔ Use CSS variables for consistent theming.
  • ✔ Comment your code and keep selectors shallow.
